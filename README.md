# 🎵 Spotify Clone

A pixel-perfect, fully responsive clone of Spotify's web interface — built with pure HTML, CSS, and JavaScript and deployed on Render.

🔗 **Live Demo:** [pavitra-spotify.onrender.com](https://pavitra-spotify.onrender.com/)

---

## 📸 Preview

![Spotify Clone Preview](https://pavitra-spotify.onrender.com/Assets/logo.svg)

---

## ✨ Features

- 🎨 **Faithful UI Recreation** — Closely mirrors Spotify's layout including the sidebar, main content area, and music player
- 🎵 **Music Player** — Bottom player bar with play/pause, previous/next controls, and a seek bar
- 📚 **Your Library** — Sidebar section with prompts to create playlists and follow podcasts
- 🔥 **Trending Songs** — Showcases a curated list of trending tracks with cover art and artist info
- 🎤 **Popular Artists** — Grid display of popular artists (Arijit Singh, A.R. Rahman, Pritam, and more)
- 💿 **Popular Albums & Singles** — Browse albums like *Aashiqui 2*, *Yeh Jawaani Hai Deewani*, and more
- 📻 **Popular Radio** — Radio stations based on popular artists
- 📊 **Featured Charts** — Weekly and daily top track charts
- 🔗 **Full Footer** — Includes navigation links for Company, Communities, Useful Links, and Spotify Plans
- 📱 **Responsive Design** — Adapts across desktop and mobile screen sizes

## 🚀 Getting Started

### Prerequisites
Just a modern web browser — no dependencies or package manager needed.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/spotify-clone.git

# Navigate into the project directory
cd spotify-clone

# Open in browser
open index.html
```

Or simply open `index.html` directly in your browser.

## 🌐 Deployment

This project is deployed on **[Render](https://render.com/)** as a static site.

To deploy your own copy:
1. Push your code to a GitHub repository
2. Go to [render.com](https://render.com/) and create a new **Static Site**
3. Connect your GitHub repo
4. Set the publish directory to the root (`/`)
5. Click **Deploy** 🚀
